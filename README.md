Notepad App

This is an Android Notepad application built with Kotlin in Android Studio. The application allows users to create, update, delete, and manage notes stored locally using SQLite.

The project includes:

- Full CRUD functionality (Create, Read, Update, Delete)

- Local data persistence with SQLite

- Multi-activity navigation

- RecyclerView implementation for dynamic note listing

- Structured and maintainable code organization

Technologies:

- Kotlin

- Android SDK

- SQLite

- RecyclerView

- XML Layouts

Project Structure

- MainActivity – Displays list of notes

- AddNoteActivity – Add new note

- UpdateNoteActivity – Edit note

- NotesAdapter – RecyclerView adapter

- NotesDatabaseHelper – SQLite database management
